  💰 비용: 50-100만원

  6. 💾 파일 출력: STL/3DM/OBJ 다운로드

  📊 피드백: 사후 확인

  3. 🧠 100ms 지연 STT + NLP 처리
  4. 📊 라이브 대시보드 실시간 업데이트
  5. 💬 즉시 피드백: 음성 품질, 전문용어, 번역
  6. 📋 실시간 요약: 핵심 포인트 즉시 정리
  7. ✅ 회의 종료 시 완전한 분석 결과 제공

  📊 업데이트: 실시간
  🌍 범위: 5개 지역 통합
