🚨 포트 연결 문제 발생

❌ 현재 상황

• Streamlit과 Flask 애플리케이션이 실행되었지만 포트에 연결할 수 없음

• 포트 8501, 8510, 8888 모두 외부 접근 불가

• Windows 방화벽 또는 네트워크 바인딩 문제로 추정

✅ 해결 방법 1: 방화벽 설정

1️⃣ Windows 방화벽 규칙 추가

관리자 권한으로 명령 프롬프트를 열고 다음 명령어 실행:

netsh advfirewall firewall add rule name="SOLOMOND_8888" dir=in action=allow protocol=TCP localport=8888
netsh advfirewall firewall add rule name="SOLOMOND_8501" dir=in action=allow protocol=TCP localport=8501
netsh advfirewall firewall add rule name="SOLOMOND_8510" dir=in action=allow protocol=TCP localport=8510
